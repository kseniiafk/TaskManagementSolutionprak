Task Management System
Микросервисное приложение для управления пользователями, проектами и задачами, разработанное на платформе .NET (C#).
Архитектурное решение
Проект построен с использованием микросервисного подхода.
Каждый компонент системы представляет собой отдельное Web API-приложение, выполняющее свою конкретную функцию.
Все клиентские запросы проходят через API Gateway, который отвечает за маршрутизацию и распределение трафика между сервисами.
Основные сервисы
ApiGateway — центральная точка входа в систему
UsersService — работа с пользователями
ProjectsService — управление проектами
TasksService — управление задачами

Структура решения

Каждый сервис является самостоятельным Web API-проектом внутри общего решения.

C#

REST API
JSON

Примеры маршрутов (через API Gateway)

GET    /api/users
POST   /api/users

GET    /api/projects
POST   /api/projects

GET    /api/tasks
POST   /api/tasks
PUT    /api/tasks/{id}
DELETE /api/tasks/{id}
Фактические маршруты могут отличаться в зависимости от конфигурации API Gateway.

Цель проекта
Данное приложение демонстрирует:
применение микросервисной архитектуры
разделение ответственности между сервисами
централизованную маршрутизацию через API Gateway
разработку распределённой системы на платформе .NET
